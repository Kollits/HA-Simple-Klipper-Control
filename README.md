# Simple Klipper Control

A Home Assistant dashboard for Klipper powered 3D printers.

Built for people who don't just own printers.

Built for people who have accidentally created small industrial robots.

## Features

Moonraker support  
Mushroom UI  
Live camera HUD  
Print progress  
Temperature monitoring  
Klipper controls  
Diagnostics  
RGB / ARGB lighting integration  
Macro launcher  

## Philosophy

Your printer isn't an appliance.

It has:
- a CPU
- sensors
- motors
- lights
- moods

Mission Control turns Home Assistant into the cockpit.

## Supported

Designed for:

- Klipper
- Moonraker
- Mainsail
- Fluidd
- BTT boards
- Voron
- RatRig
- Custom machines
